Como esse conversor funciona?

Esse conversor baseia-se no padrão IEEE 754, que define a representação e aritmética de números de ponto flutuante em sistemas computacionais. Ele descreve como os números reais são armazenados e manipulados em forma binária (0s e 1s) para permitir cálculos aproximados com números racionais.
Existem dois formatos principais definidos pelo IEEE 754: precisão simples (32 bits, este caso) e precisão dupla (64 bits). Cada formato divide os bits em três partes: sinal, expoente e mantissa (ou fração). O bit de sinal define se o número é positivo(0) ou negativo(1). O expoente determina a escala do número, enquanto a mantissa representa os dígitos significativos.
